
ІНСТРУКЦІЯ:

1. Створи Google таблицю з 4 вкладками: S1, S2, S3, S4.
2. Додай стовпці: Дата | Завдання | Час появи завдання | Час відповіді | Локація | Касир | Ім’я або номер користувача
3. Створи сервісний обліковий запис у Google Cloud, згенеруй credentials.json і додай до доступу Google таблиці.
4. Встав свій Bot Token і Spreadsheet ID у файл config.py.
5. Встанови залежності:
   pip install -r requirements.txt
6. Запусти файл:
   python main.py

Бот буде надсилати повідомлення щодня о 7:45 ранку.
